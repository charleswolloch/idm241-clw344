Triggers:

Upon mouse hover over the Creations Nav button the microinteraction is triggered. Within the list another Hover effect occurs while mouse hovers over the Creations Nav tabs.

Rules:

When mouse hovers color changes from black to grey and dropdown appears. The animation should fade in and do so over half a second. When dropdown appears user can hover over dropdown tabs. Upon hovering over the dropdown tabs the tabs change color.

Feedback:

The user upon hovering over creations should see a color change where the mouse is hovering and alist appears beneath it. Within that list appears more options for navigation and if the user hovers over any of those tabs it will have a hover effect where the color changes too.

Loops and Modes

If the user hovers over Creations the tabs are set to appear. The data set to show are the various content pages. When there is no hover no extra data is displayed.
