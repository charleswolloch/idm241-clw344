Triggers:

Upon mouse hover over a tile the microinteraction is triggered. When the mouse stops hovering the microinteraction stops getting triggered.

Rules:

When mouse hovers the front of the tile (displaying a thumbnail image and title) flips to the back and the back of the tile (displaying a description) flips to the front. When the back of the card's content appears it is covering the front of the tiles content and the same in reverse.

Feedback:

The user upon hovering over a tile sees the flip animation which shows the front information of the tile flip over to reveal the back information of the tile. The user goes from seeing an image and title to the description of the project all while hovering.

Loops and Modes:

If the user hovers over a tile the description is set to appear over the title and image. The previous information that was displayed on top moves to the bottom and the information that was on the bottom gets moved to the top. While the user continues to hover the mode continues. While there is no hover the original data is displayed.
